## BibTeX format fixes

All 25 entries had their BibTeX regenerated by
`scripts/audit-publications-fix.mjs` to match spec:

- Citation keys are now `lastnameYEARkeyword` lowercase
  (e.g. `mishra2021origindate`, was `origindate2021`).
- Authors are `Last, First and Last, First` (proper BibTeX sort form)
  rather than the previous `First Last and First Last`.
- arXiv preprints use `@article` with `journal = {arXiv preprint
  arXiv:NNNN.NNNNN}` and an `eprint`/`archivePrefix` pair so reference
  managers detect them correctly.
- Conference papers use `@inproceedings` with `booktitle` containing the
  full venue name and `pages` preserved when present.
- Journal papers use `@article` with the full journal name.
- Under-review papers use `@unpublished` with `note = {Manuscript under
  review}`.
- Origin (DATE 2021) carries `note = {Best Paper Nominee}` to match the
  award.
- Publishers are correct per venue: NSDI → USENIX Association,
  WoSC/SoCC → ACM, everything else → IEEE.
